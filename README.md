# Forex Rocket

A modern landing page for Forex Rocket – a Discord server for premium Forex trading signals. Glassmorphism dark theme with red accent. Built with Next.js 16 and Tailwind CSS.

## Features

- **Glass theme** – Frosted glass UI with backdrop blur
- **Red theme** – Bold red accent for Forex Rocket branding
- **Responsive** – Mobile-first layout with breakpoints
- **Sections** – Hero, Features, Stats, About, Events, Rules, FAQ, Footer

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Customization

Replace `https://discord.gg/your-invite` with your Discord server invite link in:

- `src/components/Navbar.tsx`
- `src/components/HeroSection.tsx`
- `src/components/Footer.tsx`

## Tech Stack

- Next.js 16 (Turbopack)
- React 19
- Tailwind CSS 3
- TypeScript
